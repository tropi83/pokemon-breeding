## Intro
Programme qui accouple les pokemons et donne les meilleures combinaisons 
et le nombre de combinaisons parfaites pour obtenir des pokémons max IV (5x31)

##  Installation
### Prérequis
- NodeJs
- TypeScript
```sh
  npm i typescript -g
```
```sh
  npm i
```

##  Utilisation
Génère 100 motisma
```sh
  tsc
  node dist/calculator.js 100 motisma
```
